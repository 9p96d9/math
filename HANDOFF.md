# HANDOFF — さわって さんすう図鑑

## 現状 (2026-07-07)
- 公開URL: https://9p96d9.github.io/math/ (GitHub Pages, masterブランチ直下)
- `index.html` = 図鑑本体（単一ファイル・依存なし）。6モジュール実装済み:
  かぞえる(10フレーム) / たしざん(ブロック合体) / 10のなかま / かけざん(アレイなぞり) / ぶんすう(ピザ) / すうちょくせん(カエルドラッグ)
- `drill.html` = 旧・単元別ドリル(小1〜中3の18単元、問題自動生成)。図鑑からリンク済み

## ゴール
「解き方を覚える前に触って感覚を掴む」算数図鑑。小さな子供が楽しく触れる、ビジュアル可読性日本一。

## デザイン原則（新モジュール追加時はこれに従う）
1. 文字より図。ひらがな、1画面1概念、説明は1行だけ
2. タッチターゲット最小44px、数字表示は4rem級ででかく
3. 操作→即座に視覚フィードバック（popアニメ既定義）
4. 配色: 背景#FFF8EE / アクセント#F76707 / 数字#2B6CB0 / 影は0 3px 0のソフト立体
5. 依存ライブラリなし・単一HTML維持（オフラインでも動く）

- [ ] かたち（図形を回転・裏返し・敷き詰め）
